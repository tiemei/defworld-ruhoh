# [grunt](http://gruntjs.com/getting-started)

grunt是javascript的任务运行器。  

# [npm](https://www.npmjs.org/)

npm是node.js的包管理器。  

## 常用命令

* `npm install`在`package.json`文件存在的目录执行，安装响应的依赖
* `npm install grunt --save-dev`将grunt的最新版本安装到工程目录，并在devDependencies中添加
